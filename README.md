# EDGE AI Surveillance and Tracking System

This project implements a real-time surveillance and tracking system
using computer vision and edge AI techniques.

The system runs on a Raspberry Pi and performs:

• Real-time person detection
• Object tracking across frames
• Repeat-sighting analytics
• Event logging and snapshot capture
• Dashboard monitoring interface

# Tech Stack
- Python
- OpenCV
- YOLO (You only look once)
- Flask
- SQLite 
- Raspberry Pi 5
